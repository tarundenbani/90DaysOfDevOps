1) Setup & Config
   git config --global user.name "username"
   git config --global user.email "email id"

2) Basic Workflow
   A) Make Directory
   B) git init : to initialize directory as git repository
   C) git status : To Check The Sattus of File Staged/Untracked/Uncommitted/Committed

3) View Changes
   git log,git status, git diff

   git log: to see the commit history
   git status : To Check The Status Of File Staged/Untracked/Tracked/Uncommitted/Commited
   git diff : To See the Changes of The File Before Committing and after Committing
